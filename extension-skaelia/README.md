# Extension Skaelia — Envoi LinkedIn direct

Cette petite extension Chrome permet, depuis l'outil de prospection
(croisia.me), d'envoyer un message LinkedIn **en un clic** dans ta propre
session LinkedIn (comme le fait Lemlist).

## Installation (une seule fois, ~1 minute)

1. Ouvre Chrome et va sur **`chrome://extensions`**
2. En haut à droite, active le **« Mode développeur »**
3. Clique sur **« Charger l'extension non empaquetée »**
4. Sélectionne le dossier **`extension-skaelia`** (celui qui contient ce fichier)
5. L'extension « Skaelia Prospection — Envoi LinkedIn » apparaît dans la liste.

Vérification : dans l'outil, ouvre **Réglages** → section « Envoi LinkedIn » doit
afficher **« installée ✓ »**. (Recharge la page croisia.me après installation.)

## Utilisation

1. Sois **connecté à LinkedIn** dans le même navigateur.
2. Dans « Mes contacts », clique sur **« Prendre contact manuellement »** puis
   **« Envoyer sur LinkedIn → »** (ou utilise « Tout prospecter »).
3. L'extension ouvre le profil, écrit le message pré-rédigé et l'**envoie
   automatiquement**. Un message de confirmation s'affiche.

## Important

- L'envoi n'est possible que vers des profils que tu peux **déjà contacter**
  (relation existante, ou messagerie ouverte). Sinon LinkedIn n'affiche pas de
  bouton « Message » et l'extension le signalera.
- LinkedIn fait évoluer ses pages régulièrement : si un envoi échoue avec
  « Élément introuvable », préviens-moi, il suffira d'ajuster l'extension.
- Reste raisonnable sur le volume : LinkedIn limite le nombre de messages par
  jour. Un envoi automatique trop massif peut faire restreindre ton compte.

## Confidentialité

L'extension n'agit que sur `croisia.me` et `linkedin.com`, et n'envoie aucune
donnée ailleurs. Tout se passe dans ton navigateur.
